Python 3.9 以上都可以執行
執行步驟
### 使用 sql_repository
1. 安裝套件
```bash
pip install -r requirements.txt
```
2. 建立 db
```bash
python use_sql_repository.py
請輸入你想要的操作 1)建立資料庫 2)新增 User Data : 1
```
3. 新增 User Data
```bash
python use_sql_repository.py
請輸入你想要的操作 1)建立資料庫 2)新增 User Data : 2
```
